# Hospital Management System

A Java-based console application designed to manage hospital operations efficiently. The system allows administrators to manage patients, doctors, appointments, and medical records through a simple command-line interface.

## Features

### Patient Management
- Add new patients
- View patient details
- Update patient information
- Delete patient records
- Search patients by ID

### Doctor Management
- Add new doctors
- View doctor information
- Update doctor details
- Remove doctors from the system
- Search doctors by ID or specialization

### Appointment Management
- Schedule appointments
- View appointment records
- Update appointment details
- Cancel appointments

### Medical Records
- Store patient medical history
- View medical records
- Update treatment information

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Collections Framework
- File Handling (Optional)
- JDBC and MySQL (Optional if database is used)

## Object-Oriented Design Principles

This project demonstrates:

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Separation of Concerns
